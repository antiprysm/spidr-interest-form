# Spidr Interest Form

A React + Tailwind CSS form for the Spidr Design coding challenge. The form collects:

- First Name
- Last Name
- Phone Number (auto-formatted)
- Email Address (validated)
- Air Fryer Cost Guess
- 16-digit Secret Spidr PIN (auto-formatted)
